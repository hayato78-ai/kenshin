# セッション管理ディレクトリ

## 概要
Claude Codeとのセッション履歴・学びを機能ごとに蓄積するai-guideファイルを管理する。

## ディレクトリ構造
```
session/
├── README.md（このファイル）
├── _template/
│   └── ai-guide-template.md（新規作成時のテンプレート）
├── excel_output/
│   └── ai-guide.md
├── csv_import/

```
